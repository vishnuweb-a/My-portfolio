# Portfolio Website Requirements (requirements.md)

# Project Name

**Vishnu Bhardwaj Portfolio 3.0**

> A premium, interactive, storytelling-based portfolio that showcases my technical expertise as a Backend Developer and AI Engineer while also serving as my personal brand, resume, blog, learning hub, and freelancing platform.

---

# Vision

Build one of the most impressive developer portfolios that instantly communicates technical depth, engineering mindset, product thinking, and creativity.

The website should not look like a template.

It should feel like exploring a modern software product.

The portfolio should be memorable enough that recruiters, startup founders, engineering managers, hackathon judges, and freelance clients remember it after visiting.
